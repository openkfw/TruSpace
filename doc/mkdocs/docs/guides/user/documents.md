# Documents

Upload, organize, and manage your documents.

## Uploading Documents

### Drag and Drop

Simply drag files into the workspace area.

### Upload Button

1. Click **"Upload"**
2. Select files
3. Wait for processing

## Supported Formats

- **Documents**: PDF, DOCX, TXT, MD
- **Spreadsheets**: XLSX, CSV
- **Presentations**: PPTX
- **Images**: PNG, JPG, GIF

## Document Actions

- **View**: Open in viewer
- **Download**: Get original file
- **Share**: Via workspace membership
- **Delete**: Remove from workspace

## Version History

Documents maintain version history in IPFS.
