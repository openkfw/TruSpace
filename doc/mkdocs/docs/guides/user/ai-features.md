# AI Features

TruSpace uses local AI for document analysis.

## AI Perspectives

After uploading a document, AI automatically generates:

- **Summary**: Key points overview
- **Analysis**: Detailed examination

## Chat with Documents

1. Open a document
2. Use the chat interface
3. Ask questions about the content
